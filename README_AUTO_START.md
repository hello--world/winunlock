# 🚀 如何自动开始完全自动化构建

## 最简单的方式

**在 Cursor 聊天中直接说：**

```
自动修复构建错误直到成功
```

就这么简单！AI 会自动处理所有步骤。

---

## 其他启动方式

### 方式 1: 使用启动脚本

```bash
npm run start-auto
```

或：

```bash
./scripts/start-auto-build.sh
```

### 方式 2: 完全自动化脚本

```bash
# 设置 GitHub Token（可选）
export GITHUB_TOKEN=your_token

# 运行
npm run auto-build "你的提交信息"
```
