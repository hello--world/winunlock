使用说明

  • 填写要用于自动解锁的 Windows 用户名和密码
  • 点击"保存配置"将设置保存到系统注册表
  • 锁定计算机（Win + L）后，锁定屏幕将显示"自动解锁"选项
  • 注意：密码以明文形式存储在注册表中，仅用于演示

⚠️ 安全警告

当前实现将密码以明文形式存储在注册表中，仅用于演示目的。

生产环境应使用更安全的方法，如：

  • Windows Credential Manager API
  • DPAPI 加密存储
  • 基于证书的身份验证
